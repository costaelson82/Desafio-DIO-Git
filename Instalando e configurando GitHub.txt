# Instalando e Configurando GitHub
## O que é GitHub?

- GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. É como uma plataforma social colaborativa, onde programadores e empresas colocam seus projetos para o desenvolvimento do código. 
- Na prática, isso significa atualizações de versões, correção de bugs, novas funções, entre outras correções. Por meio dele, é possível acompanhar qualquer alteração e quem a efetuou, além de permitir a restauração do código removido ou modificado.

- Ele suporta bem vários fluxos de desenvolvimento, é eficiênte para grandes ou  pequenos projetos. Tem compatibilidade com a maioria dos sistemas e protocolos existentes.
